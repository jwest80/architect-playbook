# Architecture Decision Records

Architecture Decision Records, or ADRs, capture important decisions, context, tradeoffs, and consequences.

## When to Write an ADR

- A decision affects architecture, security, cost, reliability, or delivery approach.
- Multiple reasonable options exist.
- Future maintainers will need to understand why a choice was made.
- A decision creates a standard or reusable pattern.
