# Azure Reference

Use this page to collect Azure notes, commands, naming conventions, service limits, and implementation links.

## Common Services

| Concept | Azure Service | Notes |
| --- | --- | --- |
| Secret storage | Azure Key Vault | Store secrets, keys, and certificates. |
| Workload identity | Managed Identity | Prefer for Azure-hosted workloads. |
| Container hosting | Azure Container Apps | Useful for containerized apps without managing Kubernetes. |
| Monitoring | Application Insights | Application telemetry, logs, metrics, and traces. |

## Useful Commands

```powershell
az account show
az group list
```
