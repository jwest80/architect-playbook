# Glossary

| Term | Definition |
| --- | --- |
| ADR | Architecture Decision Record. A short document that captures an important decision and its consequences. |
| Bicep | Azure infrastructure as code language for deploying Azure resources. |
| DDD | Domain-Driven Design. An approach to modeling software around business domains. |
| EDA | Event-Driven Architecture. A style where components communicate by publishing and consuming events. |
| IaC | Infrastructure as Code. Managing infrastructure through versioned configuration and automation. |
| Managed Identity | Azure identity feature that lets workloads authenticate to Azure services without stored credentials. |
| RAG | Retrieval-Augmented Generation. An AI pattern that retrieves relevant data before generating a response. |
