# Cloud Service Map

This table maps common architecture concepts between Azure and AWS.

| Architecture Concept | Azure | AWS |
| --- | --- | --- |
| Secret storage | Azure Key Vault | AWS Secrets Manager |
| Identity | Managed Identity | IAM Role |
| Container hosting | Azure Container Apps | ECS Fargate |
| Object storage | Azure Blob Storage | S3 |
| Queue | Azure Service Bus | SQS |
| Pub/Sub | Event Grid | EventBridge / SNS |
| Serverless functions | Azure Functions | AWS Lambda |
| Monitoring | Application Insights | CloudWatch |
| CDN/front door | Azure Front Door | CloudFront |
