# 12-Month Learning Program

This program organizes the first year of architect development into quarterly focus areas. The goal is not passive study; each focus area should produce reusable artifacts in this playbook.

## Quarterly Focus Areas

| Quarter | Focus | Target Outcomes |
| --- | --- | --- |
| Q1 | Azure, Bicep, Azure CLI, Container Apps, GitHub Actions, Key Vault, Managed Identity | Deploy secure cloud-native workloads with repeatable infrastructure and CI/CD. |
| Q2 | DDD, Clean Architecture, API Design, Messaging, Event-Driven Architecture, Security | Design maintainable service boundaries, integration contracts, and secure application architectures. |
| Q3 | AI, RAG, MCP, Agents, DevOps, Observability, AWS equivalents | Build AI-enabled systems responsibly and compare Azure patterns with AWS equivalents. |
| Q4 | Leadership, architecture reviews, mentoring, technical roadmaps | Lead architecture conversations, mentor teams, and translate technical direction into roadmaps. |
